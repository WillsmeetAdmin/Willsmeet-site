# Willsmeet Office Solutions - Next.js Website

A professional B2B website built with Next.js 14, TypeScript, and Tailwind CSS featuring a **dark theme with teal/emerald green accents**.

## 🎨 Design

- **Theme**: Dark theme (#0a0f0d background)
- **Accent**: Emerald/Teal green (#10b981)
- **Typography**: Inter (body) + Sora (headings)

## 📁 Pages

- `/` - Homepage
- `/solutions` - All solutions detailed
- `/about` - Company information
- `/contact` - Contact form & info
- `/privacy` - Privacy policy
- `/terms` - Terms of service

## 🚀 Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## 📦 Build

```bash
npm run build
npm start
```

## 🌐 Deploy to Vercel

Push to GitHub and connect to Vercel for instant deployment.
